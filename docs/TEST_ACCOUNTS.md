## 🚀 使用说明

### 1. 登录
1. 打开 `index.html` 进入登录页面
2. 选择登录方式（邮箱/手机/微信）
3. 选择登录类型（密码登录/验证码登录）
4. 输入对应的测试账号和密码/验证码
5. 点击"登录"按钮

### 2. 使用翻译功能
1. 登录成功后自动跳转到翻译器页面
2. 选择源语言（默认英语）
3. 输入要翻译的文本（最多5000字符）
4. 点击"翻译"按钮查看结果
5. 可以复制翻译结果或使用语音朗读

### 3. 查看历史记录
1. 点击顶部的"历史"标签
2. 查看所有翻译历史记录
3. 点击历史记录可以快速加载之前的翻译

### 4. 设置
1. 点击顶部的"设置"标签
2. 可以查看当前登录账号
3. 点击"退出登录"按钮退出

## 🔧 开发说明

### 本地存储
应用使用 localStorage 存储以下信息：
- `isLoggedIn`: 登录状态
- `userAccount`: 当前登录账号
- `loginTime`: 登录时间
- `translationHistory`: 翻译历史记录
- `rememberMe`: 记住我状态
- `savedAccount`: 保存的账号

### 接入真实翻译API
修改 `translator.js` 中的 `simulateTranslation` 函数，替换为真实的翻译API调用。

推荐的翻译API：
- Google Translate API
- DeepL API
- 百度翻译API
- 有道翻译API
